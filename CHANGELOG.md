# CHANGELOG

Все заметные изменения в проекте `202312-todo` фиксируются в этом файле.

Формат близок к Keep a Changelog. Версионирование в текущем состоянии проекта фактически опирается на номер из `pyproject.toml`.

## [0.1.0] - 2026-02-28

### Добавлено

- Базовый CLI для управления задачами через аргументы командной строки.
- Локальное SQLite-хранилище для таблицы `my_todo_list`.
- Операции создания, просмотра, обновления статуса и удаления задач.
- Поддержка deadline для задачи.
- FastAPI-приложение с маршрутами `/`, `/all`, `/last`, `/task/{id}`.
- Конфигурация через `src/cfg/todo_config.ini`.
- Логирование CLI в файл.
- SQL-дамп с примерными данными и скрипт инициализации базы.
- Тесты для DB-слоя, CLI и API.
- Двуязычный комплект проектной и релизной документации.

### Изменено

- README приведен к формату, который честно описывает проект как `MVP`.
- В документации явно разделены рабочий CLI-сценарий и экспериментальный API-слой.
- Лицензионные разделы и кросс-ссылки между документами приведены к единому стилю.

### Исправлено

- Нет отдельных исправлений, подтвержденных как завершенные релизные задачи в рамках этой версии.

### Примечания

- Версия `0.1.0` отражает ранний, но уже используемый рабочий контур.
- На момент оформления релизной документации воспроизводимость части тестов зависит от локальной среды, в частности от состояния `uv` cache и прав на cache-каталоги.
